 Sahayaa — AI-Based Hyperlocal Helper Platform

Problem Statement

Finding trustworthy daily helpers including maids ,cooks,nannies and caregivers is common challenge faced by urban homes.
Working couples,alone elderly persons and first-time parents finds difficulty in managing all home chores and other works because of insufficient time and trusted helpers 



 Description

In current urban cities both parents in household work,making it impossible to have enough time to handle varius household chores.
Alone elderly people need constant help with their regular tasks and first time parents also need additional help during the initial phase 
of parenting.

The present platform offers one-off service,but not a platform for trusted helpers who provide monthly help to the clients . The clients still depends on referrals from locals or agents,which  is neither safe nor secure.

There is need for a trusted ,intelligent and user friendly platform for connecting clients with verified helpers who can serve them on a daily basis and provide instant help when the user needed.

PROPOSED SOLUTION

Sahayaa-an intelligent and AI-powered platform which find helpers for users based on location,availability,budget and service category.

The platform offers the following features:

Instant helper bookings
Monthly recurring helpers
AI-based matching of users and helpers
Voice-based booking for senior citizens
Verified helper profiles for safety and trust

Tech Stack 

Frontend: Next.js (React), Tailwind CSS
Backend: FastAPI (Python)
Database: PostgreSQL
AI Matching: Machine Learning-based scoring
Payments: Razorpay
Notifications: Firebase / Twilio
Voice Support: Web Speech API
